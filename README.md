# ❤️ Heart Disease Analysis Dashboard

A **web application** that integrates and displays **interactive Tableau dashboards and stories** for analyzing heart disease data.  
The application is built using **Python, Flask, and the Tableau Embedding API v3**, allowing users to explore visual insights related to heart disease patterns and risk factors.

---

## 🚀 Features

- Interactive **Tableau dashboards and stories**
- Web-based visualization interface
- Backend built with **Flask**
- Clean frontend using **HTML, CSS, and JavaScript**
- Easy local setup and deployment

---

## 🛠️ Tech Stack

- **Backend:** Python (Flask)
- **Frontend:** HTML5, CSS3, JavaScript
- **Data Visualization:** Tableau Public (Embedding API v3)
- **Version Control:** Git & GitHub

## ▶️ How to Run the App Locally

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/heart-disease-analysis.git
cd heart-disease-analysis
```

---

### 2️⃣ Install Required Dependencies

Make sure **Python is installed**, then run:

```bash
pip install -r requirements.txt
```

---

### 3️⃣ Run the Flask Server

```bash
python app.py
```

---

### 4️⃣ Open the Application

Open your browser and go to:

```
http://127.0.0.1:5000
```
